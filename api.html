Classificazione
5 colori. Un linguaggio universale.
Ogni strada è classificata da ciclisti reali che la percorrono ogni giorno. Non algoritmi, non stime — esperienza diretta.
Nero — Score 5 — Pericolosissimo
Rosso — Score 4 — Trafficata
Giallo — Score 3 — Tranquilla
Blu — Score 2 — ZTL/Pedonale
Verde — Score 1 — Ciclabile
API Reference
Tre endpoint. Zero complessità.
JSON, CORS abilitato, nessuna autenticazione richiesta in fase beta. Pronto all'integrazione in minuti.
GET
/v1/score?lat={lat}&lon={lon}
Restituisce il danger score della strada più vicina alle coordinate. Distanza in metri inclusa.
// Esempio: Colosseo GET /v1/score?lat=41.8902&lon=12.4922 { "id": "SS-ROM-0396", "nome": "Via Sabotino", "quartiere": "Prati", "classificazione": "verde", "score": 1, "ciclabile": "si", "distanza_m": 18 }
GET
/v1/route?from={lat,lon}&to={lat,lon}
Analizza un percorso e restituisce lo score medio, lo score massimo e la lista delle strade attraversate.
// Esempio: da Prati al Colosseo GET /v1/route?from=41.8902,12.4922&to=41.9028,12.4964 { "score_medio": 3.1, "score_massimo": 4, "n_strade": 9, "strade": [ { "nome": "Via Sabotino", "score": 1 }, { "nome": "Via Cavour", "score": 4 }, ... ] }
GET
/v1/stats
Statistiche in tempo reale del database: totale strade e distribuzione per classificazione.
GET /v1/stats { "totale_strade": 549, "per_classificazione": { "nero": 39, "rosso": 197, "giallo": 178, "verde": 65, "blu": 65 } }
