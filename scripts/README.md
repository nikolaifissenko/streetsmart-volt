# Scripts — StreetSmart

Questa directory contiene (o conterrà) script per:

## Pianificati
- `csv_to_geojson.py` — converte il master CSV in GeoJSON con coordinate
- `stats.py` — genera statistiche aggregate per municipio (% rosso/nero, km ciclabili, ecc.)
- `validate.py` — controlla integrità del CSV (ID duplicati, campi mancanti, score coerente)
- `export_api.py` — genera snapshot JSON per API REST statica
